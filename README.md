# 🏨 Hotel Reviews ML Project

## Описание проекта

Данный проект посвящён анализу и подготовке данных (EDA) для задачи машинного обучения — предсказания рейтинга отеля на основе отзывов пользователей и сопутствующих признаков.

---

## Данные

Используется датасет с информацией об отелях, включающий:

* текстовые отзывы (положительные и отрицательные)
* рейтинг пользователя
* количество отзывов
* геолокацию (lat, lng)
* национальность пользователя
* теги


## Этапы анализа (EDA)

### 1. Очистка данных

* удалены выбросы (например, нереалистичное количество отзывов)
* обработаны пропуски
* удалены дубликаты

### 2. Feature Engineering

* извлечены признаки из даты (год, месяц)
* добавлена длина отзывов
* преобразованы числовые признаки из текста

### 3. Кодирование категориальных признаков

* One-Hot Encoding для:

  * отелей
  * стран
  * национальностей
* MultiLabelBinarizer для тегов

### 4. Удаление мультиколлинеарности

* удалены сильно коррелированные признаки (corr > 0.9)

---

## Модель

Используется модель:

* RandomForestRegressor

Метрика качества:

* MAPE (Mean Absolute Percentage Error)


## Масштабирование признаков

В текущей реализации масштабирование не используется, так как модель Random Forest не чувствительна к масштабу данных.
